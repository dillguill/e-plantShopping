# e-plantShopping

## Project Name: Paradise Nursery

Paradise Nursery is a React-based e-commerce application designed for plant enthusiasts. The application provides a seamless browsing and shopping experience, allowing users to explore various plant categories, manage a virtual shopping cart, and proceed toward a checkout simulation.

---

## Key Features

### 1. Navigation & Discovery

* **Landing Page:** A dedicated entry point featuring a "Get Started" button that directs users to the plant collection.
* **Responsive Navigation Bar:** Persistent links to the Home/Landing page, Product Listing, and the Shopping Cart (including a real-time item count).

### 2. Product Listing

The nursery is organized into distinct botanical sections to help users find exactly what they need:

* **Aromatic Plants:** Features species known for their fragrance, such as Lavender and Jasmine.
* **Medicinal Plants:** Features species with therapeutic properties, such as Aloe Vera and Mint.

**Product Cards include:**

* High-quality plant imagery.
* Botanical name and brief description.
* Unit price.
* **Add to Cart** functionality with visual feedback.

### 3. Shopping Cart Management

The shopping cart provides a detailed breakdown of selected items:

* **Itemized Cards:** Each plant type in the cart is displayed with its thumbnail, unit cost, and the subtotal for that specific quantity.
* **Quantity Control:** Interactive buttons to increment or decrement the number of plants.
* **Deletion:** A "Delete" option to remove a specific plant type entirely from the cart.
* **Dynamic Totals:** Real-time calculation of the total cost for all items in the cart.

---

## Navigation Actions

* **Continue Shopping:** Returns the user to the product listing page to add more greenery to their collection.
* **Checkout:** A placeholder action for finalizing the purchase.
